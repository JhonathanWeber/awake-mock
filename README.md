# Awake Web Mock

Mock funcional em Next.js para validar layout, copy e fluxo de conversão com dados falsos.

## Rodar localmente
1. Instalar dependências
2. Executar ambiente de desenvolvimento

Comandos:
- `npm install`
- `npm run dev`

## Escopo atual
- Home mock com seções principais
- Animações suaves com Framer Motion
- Dados simulados em `src/mocks/home.ts`
- Rotas do MVP: `/servicos`, `/cases` e `/contato`
- Simulação de estados `loading`, `erro` e `sucesso`
